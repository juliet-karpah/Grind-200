# Grind-200
Grind 200 is a Python-focused interview prep app that helps you practice coding questions like a real interview: write pseudocode, create your own test cases, then code. Track multiple attempts, measure your strengths and weaknesses, and optionally get AI-powered feedback.


## Key Highlights / Bullet Version

- Structured interview workflow: Pseudocode → User Test Cases → Python Code

- Hidden session timer to simulate real interview conditions without anxiety

- Multiple attempts tracked per question for reflection and improvement

- Strengths & weaknesses analytics by category and difficulty

- Optional AI critique for personalized feedback

- Python-only sandbox for safe code execution

- Focused on personal growth and learning, not competition
